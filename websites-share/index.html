一些不错的网站分享

发表于 2020-03-18 阅读次数：
本文字数： 450 阅读时长 ≈ 1 分钟

素材类

unsplash(图片分享)

一个分享免费，无版权的高质量图片的网站，可以在上面寻找壁纸，或者博客的banner。有官方api，形如https://source.unsplash.com/800x400/?sky,flower就可以得到一张800x400分辨率，内容为天空&花朵的随机图片了

Iconfont-阿里巴巴矢量图标库

可下载各种矢量图标，需要使用github账号登录

工具类

质感图标生成器精简中文版

如题，上传矢量图标后生成极具质感的图标

Syntax Highlight Code In Word Documents(复制高亮代码)

将代码转换成可以在MS Word中直接粘贴的高亮代码

Regexper(正则表达式图形化)

图形化正则表达式，方便理解和学习

convertio(转换文件格式)

将您的文件转换成任意格式

TinyPNG(压缩图片)

TinyPNG使用智能有损压缩技术将您的PNG文件的文件大小降低。

Markdown在线编辑器 MdEditor

左边源代码右边直接展示，再也不用担心不会写md文档了
